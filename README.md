### pyprogress
現在のforループの実行回数と最大実行回数から進捗度合いを視覚的に表示する。

```python3
#bar
import pyprogress
for i in range(0,20,1):
    pyprogress.bar(i,19) #現在のループカウンタの値とループの終わりを指定する。
```

### timer.py
- プログラム終了時までの実行時間を測定する。

```python3
from timer import timer
start = timer()
```

- 一部分の時間を測りたい場合。

```python3
from timer import timer
start = timer()
#startとstart.endに挟まれた部分の実行時間を測定します。
start.end()
```
